# 000 - Visión del Proyecto: Sistema de Gestión de Préstamos Bancarios

## Objetivo del Proyecto
Desarrollar una plataforma robusta y escalable que permita a los usuarios solicitar préstamos bancarios de manera eficiente, y a los administradores gestionar dichas solicitudes con total transparencia y seguridad.

## Propósito
Este sistema busca modernizar el proceso de colocación de créditos, reduciendo tiempos de respuesta y asegurando que las decisiones (aprobación/rechazo) sigan reglas de negocio claras y trazables.

## Audiencia Objetivo
1. **Clientes del Banco (Usuarios):** Personas que buscan financiamiento.
2. **Analistas de Crédito (Administradores):** Responsables de evaluar y sancionar las solicitudes.

## Alcance
- Autenticación segura mediante JWT.
- Registro de solicitudes de préstamo (monto, plazo).
- Gestión de estados (Pendiente, Aprobado, Rechazado).
- Arquitectura desacoplada (Frontend React + Backend .NET).
- Cumplimiento de estándares de seguridad OWASP.

## Valor Agregado
- Experiencia de usuario premium con React 19 y Tailwind CSS.
- Backend de alta performance con .NET 10 bajo arquitectura hexagonal.
- Mantenibilidad garantizada mediante SOLID y Clean Code.
